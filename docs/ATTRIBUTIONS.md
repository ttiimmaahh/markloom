# Attributions

Markloom stands on the work of others.

## Conversion engine

- **[MarkItDown](https://github.com/microsoft/markitdown)** (MIT) — Microsoft's
  library that does all the actual file → Markdown conversion. This project is a
  web UI and job queue around it.

## Backend

- [FastAPI](https://fastapi.tiangolo.com/) (MIT)
- [Uvicorn](https://www.uvicorn.org/) (BSD-3-Clause)
- [pydantic-settings](https://docs.pydantic.dev/) (MIT)
- [APScheduler](https://apscheduler.readthedocs.io/) (MIT)

## Frontend

- [React](https://react.dev/) (MIT)
- [Vite](https://vite.dev/) (MIT)
- [Tailwind CSS](https://tailwindcss.com/) (MIT)

- [react-dropzone](https://react-dropzone.js.org/) (MIT)

## Inspiration

- **[mazanoke](https://github.com/civilblur/mazanoke)** — a self-hosted image
  optimizer whose project structure and self-hosting ergonomics (minimal compose
  file, env-gated optional auth, docs layout) inspired this project's packaging.
